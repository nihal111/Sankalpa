# Global Hotkeys

Capture tasks from any application without switching windows.

![Global hotkeys](../gifs/gif-quick-add.gif)

## Keybindings

| Key | Action |
|-----|--------|
| `⌘⌥⌃Space` | Toggle window visibility |
| `⌘⌥⇧Space` | Open Quick Add modal |

## Quick Add Modal

The Quick Add modal provides a streamlined interface for rapid task capture:

- Task title input
- List selector (defaults to Inbox)
- Due date picker with natural language support
- Duration picker with presets (15m, 30m, 1h)
- Notes field with markdown support

## Behavior

- `⌘⌥⇧Space` opens the modal from any application
- `Escape` closes the modal without saving
- `⌘↵` creates the task and closes the modal
- New tasks are added to the selected list (Inbox by default)
