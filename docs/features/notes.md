# Task Notes

Add detailed notes to any task.

![Edit task notes](../gifs/gif-notes.gif)

## Keybinding

| Key | Action |
|-----|--------|
| `N` | Open notes editor |

## Editor

| Key | Action |
|-----|--------|
| `⌘P` | Toggle edit/preview mode |
| `⌘↵` | Save notes |
| `Esc` | Cancel and close |

- Full-screen modal for focused writing
- Markdown support with live preview
- Edit/preview toggle with `⌘P`

## Display

Tasks with notes show a note indicator icon.

## Markdown Support

Notes support standard markdown:

- Headers (`#`, `##`, `###`)
- Lists (`-`, `*`, `1.`)
- Code blocks (`` ` `` and `` ``` ``)
- Links (`[text](url)`)
- Bold/italic (`**bold**`, `*italic*`)
