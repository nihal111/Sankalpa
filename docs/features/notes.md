# Task Notes

Add detailed notes to any task.

<!-- GIF: notes.gif -->

## Keybinding

| Key | Action |
|-----|--------|
| `N` | Open notes editor |

## Editor

- Full-screen modal for focused writing
- Markdown support
- Edit/preview toggle (planned)

## Display

Tasks with notes show a note indicator icon.

## Markdown Support

Notes support standard markdown:

- Headers (`#`, `##`, `###`)
- Lists (`-`, `*`, `1.`)
- Code blocks (`` ` `` and `` ``` ``)
- Links (`[text](url)`)
- Bold/italic (`**bold**`, `*italic*`)
