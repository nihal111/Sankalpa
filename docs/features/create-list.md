# Create List

Add new lists to organize tasks.

![Create list](../gifs/gif-create-list.gif)

## Keybinding

| Key | Action |
|-----|--------|
| `Cmd+Shift+N` | Create new list |

## Behavior

- Opens inline editor in lists pane
- Press `Enter` to save, `Escape` to cancel
- New list appears at bottom of user lists
