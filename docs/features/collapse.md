# Collapse/Expand

Toggle visibility of subtasks.

![Collapse and expand](../gifs/gif-collapse.gif)

## Keybinding

| Key | Action |
|-----|--------|
| `C` | Toggle collapse/expand |

## Behavior

- Collapsed parents show `▶` indicator with child count
- Expanded parents show `▼` indicator
- Hidden children are skipped during keyboard navigation
