# Search

Find tasks across your workspace.

<!-- GIF: search.gif -->

## Keybindings

| Key | Action |
|-----|--------|
| `Cmd+Shift+F` | Global search (all tasks) |
| `Cmd+F` | Local search (current list only) — *planned* |

## Global Search

Searches across all tasks in all lists.

- Opens search modal
- Results update as you type
- Shows task title and source list
- `Enter` navigates to selected result
- `Esc` closes search

## Result Navigation

- `↑/↓` — Move through results
- `Enter` — Jump to task (switches list if needed)
- Task is selected and scrolled into view
