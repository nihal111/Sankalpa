# Reorder Tasks

Change task order within a list.

![Reorder tasks](../gifs/gif-reorder.gif)

## Keybinding

| Key | Action |
|-----|--------|
| `Alt+↑` | Move task up |
| `Alt+↓` | Move task down |

## Behavior

- Moves task among siblings at same depth
- Collapsed parents move with their subtree
- Depth adjusts automatically when crossing hierarchy boundaries
