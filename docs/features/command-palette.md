# Command Palette

Quick access to all available commands.

<!-- GIF: command-palette.gif -->

## Keybinding

| Key | Action |
|-----|--------|
| `Cmd+K` | Open command palette |
| `↑/↓` | Navigate commands |
| `Enter` | Execute selected command |
| `Esc` | Close palette |

## Behavior

- Shows commands available in current context
- Fuzzy search filters commands as you type
- Each command shows its keybinding
- Commands are grouped by category

## Context Awareness

The palette shows different commands based on:

- Current pane (lists vs tasks)
- Selection state (single vs multi-select)
- Task properties (has subtasks, is collapsed, etc.)
