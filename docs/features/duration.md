# Duration

Estimate time required for tasks.

![Set duration](../gifs/gif-duration.gif)

## Keybinding

| Key | Action |
|-----|--------|
| `⌥D` | Open duration picker |

## Input

The picker offers presets and custom input:

**Presets:**
- 15m, 30m, 45m
- 1h, 1.5h, 2h
- 3h, 4h, 8h

**Custom input:**
- `20m`, `90m`
- `2.5h`, `3h`

## Display

Tasks show a ruler icon with duration text (e.g., "1h 30m").

## Clearing

Select "None" or clear the input to remove duration.
